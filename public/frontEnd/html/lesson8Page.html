1 - Scripts

Are pieces of code that are executed on a web page. Are written in JavaScript and are inserted within an HTML file using the script tag

Scripts can be used for various functions as adding dynamic content, manipulate data, create APIs, etc

scripts can be ran in various ways as inserting them in a html file, whithin the script tag, presenting a source path to another file in html, whith the cmd after installing node.js or in the browser console



2 - Enviroment variables natively in Node.js

To define an environment variable in Node.js we can:

-> Set the environment variable directly in the command line when running the Node.js application

("PORT=3000 node app.js")

This will set the port to 3000 and run app.js

-> set the environment variables in a .env file in the root directory

(" PORT=3000 DB_USER=myusername DB_PASS=mypassword" )

where the format is VARNAME = value

("npm install dotenv")

then instal the dotenv package

in js

("require('dotenv').config();")

("const port = process.env.PORT")

this will return the port



3 - Forms

To start up we have to use HTML to create a form that includes input fields

("form action="/submit-form" method="POST" label for="name">Name: /label input type="text" id="name" name="name" /form ")

then in app.js

(" const express = require('express'); const app = express(); app.post('/submit-form', (req, res) => { code }); ")

here, the path correspond to the action of the form

(" const bodyParser = require('body-parser'); app.use(bodyParser.urlencoded({ extended: true })); app.post('/submit-form', (req, res) => { const name = req.body.name; /*use the form data*/ }); ")

body-parser to parse the form data from the request body, and then assigns the values of the name

5 - Creating components and pass props

Components can be created by defining functions that encapsulate some logic and return a piece of user intreface

(" function Button(props) { return `button style="background-color: ${props.color}"> ${props.label} /button>`; } ")

("const newButton = Button({ color: "red", label: "Click me!" });")

("console.log(newButton);")

we show button in the console by logging the newButton variable. This will output the string of HTML representing the button
