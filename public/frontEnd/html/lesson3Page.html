1 - How to loop in arrays in javascript

Basic loop

(" const numbers = [1, 2, 3, 4, 5]; for (let i = 0; i < numbers.length; i++) { console.log(numbers[i]); } ")

map() method -> Is an array method that allowsto create a new array by calling a function on each element of the original array

("' const numbers = [1, 2, 3, 4, 5]; const doubledNumbers = numbers.map((num) => num * 2); console.log(doubledNumbers); ")

filter() method -> Is an array method that allows to create a new array with elements that return true from the function

(" const numbers = [1, 2, 3, 4, 5]; const evenNumbers = numbers.filter((num) => num % 2 === 0); console.log(evenNumbers); ")

reduce() method -> Is an array method that allows to reduce an array to a single value by executing a function on each element

(" const numbers = [1, 2, 3, 4, 5]; const sum = numbers.reduce((acc, num) => acc + num, 0); console.log(sum); ")

Note that is usually better to make loops is functions, in javascript

This is becaouse the code is more readablle, the loop can be reused, the code has can be easier divided and structured and it is easier to find and fix bugs

3 - Security need for serving static files in Express

Use the express.static to serve static files. This automatically sets the correct MIME type for each file and prevents directory attacks.

Store static files in a separate directory that is not accessible to the public. Files cant be accessed by manipulating the URL

Use a Content Security Policy (CSP) to prevent content injection attacks. A CSP is a security mechanism that allows you to control which resources can be loaded by your application.

Set the Content-Security-Policy header in the HTTP response which tells what can resources can be loaded by the application
