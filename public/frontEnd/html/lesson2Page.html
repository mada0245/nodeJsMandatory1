1 - Why should we always use const or let instead of var ?

They have more predictable behavior and help to make the code more readable and maintainable

Problems with var:

Function scoping: Variables declared with var are function-scoped, they can be accessed in the entire function where they are declared. They can be hard to understand where are being used which can bring bugs

They are on the top of the scope, they can be accessed before are declared

They can be reassigned and have theier value changed, using const this problem will not happen or let if values have to be changed
