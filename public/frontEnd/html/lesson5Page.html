1 - Import and export

Import and export are used in frontend to modularize code and manage dependencies

The import statement will pull a module or a functionality from another file or library

A function can be exported with export() which makes it available to use in other parts of the code when it is imported



2 - Importing and exporting in node.js

First it is necessarry to add "type": "module" in the package.json file to enable modules in the Node.js project

Export

(" export function functionName() { /*code*/ } ")

Import

(" import { functionName } from './modulePath.js'; ")

Run the code with the --experimental-modules flag

This tells Node.js to use ES modules instead of the CommonJS module



3 - XSS

Is a security vulnerability that allows to inject malicious scripts into a website or web application

which can get access to information or take control browser

How to avoid it:

Add input validation on all user-generated content that is sent to the server

Encode output made by user

Use secure HTTP-only cookies

Use content security policy (CSP)
