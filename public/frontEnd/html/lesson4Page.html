Fetch

GET

(" fetch('https://example.com/data') .then(response => response.json()) .then(data => console.log(data)) .catch(error => console.error(error)); ")

PUT

(" fetch('https://example.com/data/123', { method: 'PUT', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ name: "firstName secondName", age: 26 }) }) .then(response => response.json()) .then(data => console.log(data)) .catch(error => console.error(error)); ")



Getting the date in javascript

(" const currentDate = new Date(); console.log(currentDate); ")

This returns the current date, hour and hour region

Is also possible to get part of the date using the date methods as so:

(" const currentDate = new Date(); const currentYear = currentDate.getFullYear(); console.log(currentYear); ")

This will return the current year
