1 - This is how to use git in a project:

("mkdir folderName") -> Create a new project folder

("cd folderName") -> Go to project directory

("git init") -> Initialize a Git repository

Create a new repository on github and copy the repositoryURL

("git remote add origin repositoryURL") -> Add the remote repository to your local Git repository

("touch fileName" + ".fileExtention") -> Create a file

("git add filName" + ".fileExtention" || "git .") -> Stage the changes of a file or all files in the directory

("git commit -m "message"") -> Commit the staged changes

("git push -u origin main") -> Push changes

4 - This is how to design a REST API with express in Node.js

("npm init -y " , "npm install express")

make a js file file named app

In app.js :

("const express = require("express");")

("const app = express();")

(" app.get("/link", (req, res) => { res.send({ /*code*/ }); }); ") -> get something from the server

(" app.get("/link/:id", (req, res) => { /*code*/ res.send({ /*response code*/ }); }); ") -> get something from the server by id

(" app.post("/link/post", (req, res) => { /*code*/ res.send({ /*response code*/ }); }); ") -> post something to the server

(" app.patch("/link/:patch", (req, res) => { /*code*/ res.send({ /*response code*/ }); }); ") -> patch something from the server by id

(" app.delete("/link/:delete", (req, res) => { /*code*/ res.send({ /*response code*/ }); }); ") -> delete something from the server by id

(" app.listen(8080, () => { console.log("Server is running on port", 8080); }); ") identify the server port and print he message when it is running
